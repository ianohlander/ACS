# WorldTree SVG UX/UI Asset Package

This package contains production-oriented SVG UI assets generated from the WorldTree UI SVG specification.

## Included Themes
- Paper UI: parchment, ink, hand-drawn imperfections, soft shadows.
- Dark Fantasy UI: gothic stone/metal, sharp borders, glow states, rune styling.

## Structure
assets/paper and assets/dark-fantasy contain backgrounds, panels, buttons, borders, dividers, icons, and textures.

## Notes
Each SVG is standalone and includes its own filters or definitions where needed. Button states are exported separately.
