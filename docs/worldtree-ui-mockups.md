# WorldTree UI Design Concepts
## Living Engine for Creating Worlds — Visual Mockup Guide

---

## Design Philosophy

The WorldTree interface embraces a **living, organic aesthetic** that reflects the core metaphor: worlds grow from seeds, extend through roots, branch into paths, and bloom with content. Every UI element should feel alive — subtle animations, organic shapes, and natural color progressions that echo real botanical growth.

### Core Visual Language

| Element | Treatment |
|---------|------------|
| **Primary Colors** | Deep forest green `#1F3D2B`, vibrant growth green `#2ECC71`, midnight blue `#0B1E2D` |
| **Accent Colors** | Electric cyan `#00D4FF`, warm gold `#FFD700` |
| **Shapes** | Organic curves, branching patterns, circular nodes, flowing connections |
| **Typography** | Modern sans-serif (primary), optional light serif (headings) |
| **Motion** | Subtle growth animations, pulsing connections, leaf-fall transitions |

## Screen 1: World Seeding (Project Creation)

### Option 1 — The Seed Vault
A subterranean chamber filled with glowing orbs — each orb is a potential world seed. User selects an orb to begin.

- **Visual**: Dark cavern with bioluminescent seed orbs floating in fluid-like clusters
- **Interaction**: Hover expands orb revealing world type (fantasy, sci-fi, horror); click plants the seed
- **Atmosphere**: Mysterious, ancient, discovery-focused

### Option 2 — The Nursery Bed
A warm, sunlit greenhouse with seed packets arranged in raised beds. Each packet represents a world template.

- **Visual**: Soft golden light filtering through glass ceiling; packets labeled with genre icons
- **Interaction**: Drag packet to planting zone to create world; soil particles animate on hover
- **Atmosphere**: Welcoming, nurturing, beginner-friendly

### Option 3 — The Cosmic Orchard
A starfield backdrop with fruit-like worlds hanging from glowing branches. User plucks a world to begin.

- **Visual**: Night sky with constellations; glowing fruits each represent a world archetype
- **Interaction**: Reach/click to harvest world template; stars trail behind cursor
- **Atmosphere**: Expansive, magical, infinite possibility

### Option 4 — The Root Network
Underground view of vast root systems where seeds are planted. Tendrils show existing worlds.

- **Visual**: Cross-section of earth showing root networks; new seeds planted in empty spaces
- **Interaction**: Click empty soil patch to plant new seed; roots visually connect to parent world
- **Atmosphere**: Foundational, structural, power-focused

### Option 5 — The Sprouting Terminal
Clean, modern interface where a single seed rests in a growth chamber. Minimalist but alive.

- **Visual**: White/light background with central seed pod; subtle green glow pulses
- **Interaction**: Click seed to expand into world options; stem grows upward as selection progresses
- **Atmosphere**: Focused, streamlined, professional

## Screen 2: Root Engine (Core Systems Dashboard)

### Option 1 — The Underground Lab
A tech-botanical fusion control room with organic screens growing from wall-vines.

- **Visual**: Dark green ambient lighting; monitors are leaf-shaped displays embedded in living walls
- **Interaction**: Touch vine-stems to access system modules; data flows like sap through visible tubes
- **Atmosphere**: Advanced, integrated, living technology

### Option 2 — The Tree Heart
Central trunk cross-section showing rings of system functionality. Each ring = one system layer.

- **Visual**: Giant tree trunk cross-section with concentric rings; inner rings = core, outer = extensions
- **Interaction**: Click ring segment to expand system config; rings glow on selection
- **Atmosphere**: Foundational, hierarchical, architectural

### Option 3 — The Root Map
Topographic map view where systems are root-branches spreading across the world.

- **Visual**: Stylized terrain map with root-paths connecting system nodes; terrain rises/falls with complexity
- **Interaction**: Drag to pan; click nodes to configure; roots pulse with data flow
- **Atmosphere**: Spatial, navigational, complex-system visualization

### Option 4 — The Laboratory Greenhouse
Clean white space with central holographic tree. Systems are leaf-clusters attached to branches.

- **Visual**: Minimalist lab aesthetic; holographic tree shows system hierarchy; leaves = modules
- **Interaction**: Drag leaves to reorder; click to expand; wilted leaves = disabled systems
- **Atmosphere**: Clinical, organized, modular

### Option 5 — The Ancient Observatory
Steampunk-botanical control room with brass instruments and living gauges. Victorian meets bio-tech.

- **Visual**: Dark wood and brass console; plant vines wrap around gears; gauges are flower-meters

## Screen 4: Leaf Layer (Content/Asset Manager)

### Option 1 — The Autumn Grove
A forest floor covered in fallen leaves — each leaf is an asset. User rakes through to find.

- **Visual**: Golden autumn forest; leaves of many colors blanket the ground; some glow
- **Interaction**: Rake/swipe to reveal assets; drag leaves to categorize; glowing = favorites
- **Atmosphere**: Organized chaos, treasure-hunting, tactile

### Option 2 — The Canopy Gallery
Upper tree canopy where assets hang like ornaments. Sunlight filters through leaf-collections.

- **Visual**: Bright sun-dappled canopy; assets displayed as glowing ornaments on branches
- **Interaction**: Reach/click to grab; drag to organize; shake branch to reveal hidden assets
- **Atmosphere**: Bright, display-focused, curated

### Option 3 — The Herbarium
Museum-style display cases with preserved botanical specimens — each specimen = asset type.

- **Visual**: Clean museum interior; glass cases with labeled specimens; botanical illustrations
- **Interaction**: Open cases to browse; magnifying glass for detail; label-maker for tagging
- **Atmosphere**: Categorized, scholarly, systematic

### Option 4 — The Seed Bank
Cold-storage vault with rows of seed packets — each packet = asset category. Frost on glass.

- **Visual**: Industrial cryo-storage; frozen mist; labeled drawers and packets
- **Interaction**: Pull drawer to reveal; thaw to preview; pack seeds for export
- **Atmosphere**: Technical, inventory-focused, cold storage

### Option 5 — The Living Press
Vintage printing workshop where assets are printed onto leaves. Press prints, leaves become assets.

- **Visual**: Steampunk print shop; brass presses; drying racks full of printed leaves
- **Interaction**: Load press with asset; crank to print; collected leaves = assets
- **Atmosphere**: Craft, production, tangible output

## Screen 5: Forest Hub (World Gallery/Sharing)

### Option 1 — The Great Forest
Vast 3D forest where each tree is a world. User flies through to explore.

- **Visual**: Epic scale forest; each tree-crown = world thumbnail; fog and light rays
- **Interaction**: Fly/camera through forest; land on trees to preview; plant new tree to publish
- **Atmosphere**: Infinite, explorable, grand

### Option 2 — The Trading Post
Frontier town square with stalls — each stall = a world showcase. Traders = creators.

- **Visual**: Western-themed town square; colorful stalls with world previews; creator avatars
- **Interaction**: Browse stalls; trade/world-swap with creators; chat with traders
- **Atmosphere**: Community, commerce, social

### Option 3 — The Star Map
Celestial interface where worlds are stars in constellations. Constellations = genres.

- **Visual**: Dark space with star-cluster constellations; each star = world; nebulae = categories
- **Interaction**: Zoom into constellations; click stars for preview; draw new constellations
- **Atmosphere**: Astronomical, categorization, discovery

### Option 4 — The Tree Museum
Grand museum interior where world trees are displayed in exhibition halls. Curated galleries.

- **Visual**: Elegant museum; world-trees in glass cases; plaque descriptions; spotlight lighting
- **Interaction**: Walk gallery; read plaques; rate/favorite; submit for featured exhibition
- **Atmosphere**: Curated, prestigious, quality-focused

### Option 5 — The Festival Grounds
Carnival/festival grounds with world-themed rides and attractions. Each attraction = world to try.

- **Visual**: Vibrant festival; ferris wheel with world-cars; game booths = featured worlds
- **Interaction**: Ride attractions to experience worlds; win prizes for engagement; join parade
- **Atmosphere**: Fun, gamified, celebration
