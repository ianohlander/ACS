From Milestone 10 forward, each milestone records what changed, how to compare it, and which commit/tag represents that version.

Milestone 20: Exits, Portals, And Map Graphs

Status: complete after commit and tag creation.

Git tag: milestone-20

Primary comparison target: compare milestone-19 with milestone-20 to see map travel become editable data instead of hardcoded sample wiring.

  • Added Exits & Portals layer mode in Map Workspace.
  • Added editor-core exit list/upsert/delete helpers.
  • Added selected-map exit summaries and derived map graph summaries.
  • Updated runtime/editor labels, User Guide, System Reference, screenshots, PDFs, and quality cleanup notes.

Milestone 19: Map Context And Classified Libraries

Status: complete after commit and tag creation.

Git tag: milestone-19

Primary comparison target: compare milestone-18 with milestone-19 to see map selectors move into work panels and library data become classified objects.

  • Added direct map selectors in Map Workspace and Logic.
  • Added library category, skill, trait, spell, flag, and custom object data structures.
  • Migrated entity profiles from freeform skill strings to definition-backed skill ids.
  • Updated validation, sample content, User Guide, System Reference, screenshots, and PDFs.

Milestone 18: Focused Editor Workspaces

Status: complete after commit and tag creation.

Git tag: milestone-18

Primary comparison target: compare milestone-17 with milestone-18 to see the editor change from an all-panels dashboard into focused authoring workspaces.

  • Added workspace navigation state for Adventure, World Atlas, Map Workspace, Libraries, Logic, and Test & Publish.
  • Added section metadata so each workspace shows only relevant panels while editing the same AdventurePackage draft.
  • Added direct URL hash support for focused editor areas and immediate sample-content rendering before async draft/API bootstrap.
