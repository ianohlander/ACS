ACS Version Trail

Version History

From Milestone 10 forward, each milestone records what changed, how to compare it, and which commit/tag represents that version.

Milestone 18: Focused Editor Workspaces

Status: complete after commit and tag creation.

Git tag: milestone-18

Primary comparison target: compare milestone-17 with milestone-18 to see the editor change from an all-panels dashboard into focused authoring workspaces.

  • Added workspace navigation state for Adventure, World Atlas, Map Workspace, Libraries, Logic, and Test & Publish.
  • Added section metadata so each workspace shows only relevant panels while editing the same AdventurePackage draft.
  • Added direct URL hash support for focused editor areas and immediate sample-content rendering before async draft/API bootstrap.
  • Updated app labels, roadmap, user guide, system reference, screenshots, and PDFs for Milestone 18.
